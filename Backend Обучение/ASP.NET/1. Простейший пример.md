Рассмотрим код, который создаёт простейшее приложение:
```cs
var builder = WebApplication.CreateBuilder(args);

var app = builder.Build();

app.MapGet("/", () =>"Hello World!");

app.Run();
```
Это так называемое Minimal API - упрощённая минимизированная модель для запуска веб-приложения ASP.NET.

Приложение (app) в ASP.NET Core представляет объект `Microsoft.AspNetCore.Builder.WebApplication`. Это приложение настраивает всю конфигурацию приложения: маршруты, зависимости и тд.
Для создания такого приложения требуется специальный класс-строитель WebApplicationBuilder. В начале файла `Program.cs` создаётся такой объект с помощью статического метода 
```cs
var builder = WebApplication.CreateBuilder(args);
```
Получив объект `WebApplicationBuilder`, вызываем метод Build(), который возвращает объект `WebApplication`. 
```cs
var app = builder.Build();
```
С помощью объекта `WebApplication` можно настроить всю инфраструктуру приложения - конфигурацию, маршруты и тд. 

Метод `app.MapGet()` в качестве первого параметра принимает путь, по которому можно обратиться к приложению. В данном случае это путь "/", то есть корень веб-приложения.
```cs
app.MapGet("/", () =>"Hello World!");
```
В конце необходимо запустить приложение. Для этого используем метод `app.Run()`.
```cs
app.Run();
```
